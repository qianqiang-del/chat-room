# 💬 TCP 网络聊天室

一个基于 Go 语言原生 net 包实现的简单 TCP 聊天室。支持多用户同时在线、昵称唯一性、消息广播、加入/离开通知等功能。

## ✨ 功能特点

- 用户使用**唯一昵称**登录
- 昵称重复时服务端会提示重新输入
- 用户加入或离开聊天室时，**广播通知**所有在线用户
- 用户发送的消息**实时广播**给其他所有人（不包括自己）
- 服务端支持**并发处理**多个客户端连接
- 客户端输入 `exit` 即可退出聊天室
