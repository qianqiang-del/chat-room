# 💬 TCP 网络聊天室

一个基于 Go 语言原生 net 包实现的简单 TCP 聊天室。支持多用户同时在线、昵称唯一性、消息广播、加入/离开通知等功能。

## ✨ 功能特点

- 用户使用**唯一昵称**登录
- 昵称重复时服务端会提示重新输入
- 用户加入或离开聊天室时，**广播通知**所有在线用户
- 用户发送的消息**实时广播**给其他所有人（不包括自己）
- 服务端支持**并发处理**多个客户端连接
- 客户端输入 `exit` 即可退出聊天室

## 🚀 快速开始

### 环境要求

- Go 1.16 及以上版本（使用标准库，无需额外依赖）

### 1. 克隆仓库

```bash
git clone https://github.com/qianqiang-del/chat-room.git
cd chat-room
2. 启动服务端
bash
go run main.go
服务端将监听 localhost:8080，并输出 服务器开始监听。

3. 启动客户端（可开启多个终端）
bash
go run client.go
按照提示输入昵称（例如 Alice），成功后即可开始发送消息。

4. 发送与接收消息
在客户端输入任意文本，按回车，消息会广播给其他所有在线用户

输入 exit 即可退出聊天室，并通知其他用户

📁 项目结构
text
.
├── main.go          # 服务端代码（TCP 监听、用户管理、广播）
├── client.go        # 客户端代码（昵称注册、消息收发）

🧠 技术要点
TCP 协议：使用 net.Listen 和 net.Dial

并发模型：每个客户端一个 goroutine，全局 map 存储在线用户

同步：sync.Mutex 保护共享 map

广播：通过 channel 实现异步消息分发

协议：简单换行符 \n 分隔消息
